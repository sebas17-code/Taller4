# Taller 4 - Ecosistema Web Bootstrap
## Champions League 2025/26

Este proyecto es el taller final de Bootstrap 5 hecho para el SENA.

### Tema
UEFA Champions League 2025/26

### Paginas
- `index.html` - Landing page principal
- `app/login.html` - Inicio de sesion
- `app/registro.html` - Registro de club
- `app/recuperar.html` - Recuperar contraseña
- `app/admin.html` - Panel administrativo
- `app/cliente.html` - Portal del club

### Tecnologias usadas
- Bootstrap 5.3.3 (via CDN)
- Bootstrap Icons 1.11.3
- HTML5

### Como usar
Abrir el archivo index.html en el navegador o desplegarlo con GitHub Pages.

### Estructura del proyecto
```
taller4/
  app/
    login.html
    registro.html
    recuperar.html
    admin.html
    cliente.html
  assets/
    img/
```
